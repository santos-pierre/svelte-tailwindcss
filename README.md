# Svelte + Tailwind :heart_eyes:

## Vite Installation - Svelte

Vite Setup

```bash
npm create vite@latest

npm install
```

1. Name Your Project.
2. Choose Svelte.
3. Choose Svelte => Javascript / Svelte-TS => Typescript.

## Tailwind CSS Installation

Let Svelte do the work !

```bash
npm i svelte-add

npx svelte-add tailwindcss

npm install

npm run dev
```

Enjoy :tada:
